## Kullanılan API
OpenWeather https://openweathermap.org/api

## Projeyi Nasıl Çalıştırabiliriz?
1. Terminale geçip proje klasörüne gitmek için şunu yazın:
cd havadurumu
2. Gerekli paketi kurmak için:
npm install
3. Projeyi başlatın:
npm run dev
4. Son olarak şu adrese gidin:
http://localhost:5173
